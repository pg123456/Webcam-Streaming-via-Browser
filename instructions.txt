Requirements:
-1 server
-1 host-client
-1 or more viewer-clients

Instructions:
1. Server: Download and install NodeJS if it hasn't already been done before.
2. Server: In the command line, navigate to the folder containing server.js and run the command "node server.js"
3. Host: In the browser, navigate to "(server's IP address):8080" and click "Record"
4. Client(s): In the browser, navigate to "(server's IP address):8080" and click "View"
